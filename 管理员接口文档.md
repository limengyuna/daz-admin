# 管理员后台 API 接口文档

> 基础地址：`http://localhost:8080`（通过网关） 或 `http://localhost:8084`（直连）
>
> 除登录接口外，所有接口均需在请求头携带：`Authorization: Bearer {token}`
>
> 无权限统一返回：`{"code": 401, "message": "未登录或无管理员权限", "data": null}`

---

## 一、管理员认证

### 1. 管理员登录

- **请求方式：** `POST /api/admin/auth/login`
- **Content-Type：** `application/json`
- **是否需要 Token：** ❌ 不需要

**请求体：**

```json
{
  "username": "admin",
  "password": "admin123"
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | String | 是 | 管理员账号 |
| password | String | 是 | 密码 |

**成功返回：**

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "adminId": 1,
    "username": "admin",
    "nickname": "超级管理员",
    "avatarUrl": null,
    "role": 1,
    "token": "eyJhbGciOiJIUzI1NiIs..."
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| adminId | Long | 管理员ID |
| username | String | 登录账号 |
| nickname | String | 昵称 |
| avatarUrl | String | 头像链接 |
| role | Integer | 角色：1-超级管理员，2-普通管理员 |
| token | String | JWT Token，后续请求需携带 |

**失败返回：**

```json
{
  "code": 500,
  "message": "账号或密码错误",
  "data": null
}
```

## 二、仪表盘数据统计

### 2. 获取仪表盘数据

- **请求方式：** `GET /api/admin/dashboard`
- **是否需要 Token：** ✅ 需要

**请求示例：**

```
GET /api/admin/dashboard
Headers:
  Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
```

**返回：**

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "totalUsers": 156,
    "totalActivities": 83,
    "totalMoments": 210,
    "todayNewUsers": 5,
    "todayNewActivities": 3,
    "todayNewMoments": 12,
    "recruitingActivities": 28,
    "bannedUsers": 2
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| totalUsers | Long | 用户总数 |
| totalActivities | Long | 活动总数 |
| totalMoments | Long | 动态总数 |
| todayNewUsers | Long | 今日新增用户 |
| todayNewActivities | Long | 今日新增活动 |
| todayNewMoments | Long | 今日新增动态 |
| recruitingActivities | Long | 招募中的活动数 |
| bannedUsers | Long | 被封禁的用户数 |

## 三、用户管理

### 3. 获取用户列表（分页 + 搜索）

- **请求方式：** `GET /api/admin/users`
- **是否需要 Token：** ✅ 需要

**请求参数（Query）：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | int | 否 | 0 | 页码，从0开始 |
| size | int | 否 | 10 | 每页数量 |
| keyword | String | 否 | - | 按用户名/昵称模糊搜索 |

**请求示例：**

```
GET /api/admin/users?page=0&size=10&keyword=张三
```

**返回：**

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "userId": 1,
        "username": "zhangsan",
        "nickname": "张三",
        "avatarUrl": "https://oss.xxx.com/avatar/1.jpg",
        "gender": 1,
        "city": "北京",
        "creditScore": 100,
        "status": 1,
        "createdAt": "2025-12-01T10:30:00"
      }
    ],
    "total": 156,
    "page": 0,
    "size": 10,
    "totalPages": 16

}
```

## 四、活动管理

### 7. 获取活动列表（分页 + 搜索 + 状态筛选）

- **请求方式：** `GET /api/admin/activities`
- **是否需要 Token：** ✅ 需要

**请求参数（Query）：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | int | 否 | 0 | 页码，从0开始 |
| size | int | 否 | 10 | 每页数量 |
| keyword | String | 否 | - | 按标题模糊搜索 |
| status | Integer | 否 | - | 状态筛选：0-招募中，1-已满员，2-已结束，3-已取消 |

GET /api/admin/activities?page=0&size=10&keyword=火锅&status=0

        "categoryIds": "[1,3]",

        "endTime": "2026-03-22T21:00:00",
        "maxParticipants": 4,
        "paymentType": 1,
        "status": 0,

        "initiatorId": 1,
        "initiatorNickname": "张三",
        "initiatorAvatar": "https://oss.xxx.com/avatar/1.jpg",
        "currentParticipants": 2
      }
    ],
    "total": 83,
    "page": 0,
    "size": 10,

**活动状态说明：**

### 9. 恢复活动
